

# 📄 Login Page (HTML & CSS)

This project is a basic **Login Page** created using only **HTML and CSS**.
It demonstrates the structure of a web form and styling using CSS.

🔗 **Live Demo:**
[https://shyamz2606-loginpage.netlify.app/](https://shyamz2606-loginpage.netlify.app/)

---

## 📌 Project Overview

* Built using **pure HTML & CSS**
* No JavaScript used
* Focus on **form structure and styling**
* Suitable for beginners learning frontend basics

---

## 🛠️ Technologies Used

* **HTML5** → Page structure
* **CSS3** → Styling and layout

---

## 📂 File Structure

```
login-page/
│── index.html
│── style.css
│── screenshot.png
```

---

## 📜 HTML Details

The HTML file includes:

* `<form>` element for login input
* `<input>` fields:

  * Username / Email
  * Password
* `<button>` for submit
* Proper use of:

  * `<div>` for layout
  * `<label>` for accessibility




## 🎨 CSS Details

The CSS file handles:

* Layout alignment (centering form)
* Spacing and margins
* Input and button styling
* Font and colors
